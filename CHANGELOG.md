## Feature Highlights

### Sleep Calculation Logic
- Calculates bedtime by subtracting (cycles × 90 minutes + 15 minutes) from wake-up time
- Generates 3-6 cycle options (4.5-9 hours)
- Highlights 5-6 cycles as recommended for optimal rest

### User Experience
- Pre-filled time input with sensible default (current time + 8 hours)
- Calculate on button click or Enter key press
- Smooth scrolling to results
- Staggered card animations for visual appeal
- Clear visual hierarchy and feedback
